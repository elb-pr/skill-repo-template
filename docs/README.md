## Adding documents

1. Convert any PDFs or Office files to Markdown via `markitdown`
2. Place in the correct `docs/<category>/` subfolder
3. Use kebab-case filenames with a source prefix where relevant (e.g. `source-topic.md`)
4. Rebuild the index: `cd scripts && python3 build_index.py`

## Retrieval

[`_index.xml`](./_index.xml) is generated by `scripts/build_index.py`. It uses:

- **BM25** — Okapi BM25, k1=1.5, b=0.75
- **TF-IDF** — normalised
- **RRF** — Reciprocal Rank Fusion, k=60
- **Chunking** — H1→H2→H3 heading hierarchy; paragraph breaks as secondary

Each chunk carries `start_line`/`end_line`, `<description>`, `<categories>`, per-term scores, and RRF rank.
